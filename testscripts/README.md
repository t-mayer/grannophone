# Grannophone test and demo scripts

This directory contains some test and demo scripts.

They can be used to test GPIO button functions, controlling LEDs via GPIO, displaying fullscreen messages on the TFT (for demos/mockups), etc.
None of these scripts are needed for regular/production use.
